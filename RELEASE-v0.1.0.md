# v0.1.0

First public release of Screen Time Tracker.

## What is included

- Active computer-time tracking
- Daily limits for study, socials, and browser fun
- Browser-aware classification with optional extension
- Tray-first workflow with quick glance
- Review and health tools for improving categorization
- Weekly review with calendar heatmap
- Focus mode and hard block mode
- Analytics, timeline, exports, and custom categories

## Notes

- Windows only
- PowerShell + WinForms app
- All usage data stays local
- Runtime data is stored under `data/` and is not meant to be committed

## Recommended next steps

- Add screenshots to the repository page
- Refine your own rules and categories after first launch
- Enable autostart if you want full-day tracking
